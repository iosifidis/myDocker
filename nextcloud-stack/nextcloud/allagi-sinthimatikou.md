# Αλλαγή συνθηματικού του admin

Μπορείς να χρησιμοποιήσεις την εντολή `occ` του Nextcloud για να διαχειριστείς τους χρήστες και να αλλάξεις τον κωδικό πρόσβασης. Η εντολή `occ` πρέπει να εκτελεστεί μέσα στο container του Nextcloud.

Ακολουθούν τα βήματα:

1.  **Σύνδεση στο Nextcloud container:**
    Θα πρέπει να συνδεθείς στο container του `app` του Nextcloud.

    ```bash
    docker exec -it nextcloud_app sh
    ```
    Αυτό θα σε βάλει σε ένα shell μέσα στο container.

2.  **Έλεγχος των χρηστών:**
    Για να δεις όλους τους χρήστες (συμπεριλαμβανομένου του διαχειριστή), μπορείς να χρησιμοποιήσεις την εντολή `user:list`:

    ```bash
    php occ user:list
    ```
    Αυτή η εντολή θα σου εμφανίσει μια λίστα με τους χρήστες και τους ρόλους τους. Ο διαχειριστής θα είναι συνήθως ο πρώτος χρήστης που δημιουργήθηκε κατά την εγκατάσταση.

    Μια πιο πλήρης εικόνα του Nextcloud.
    

3.  **Αλλαγή κωδικού πρόσβασης διαχειριστή:**
    Αφού βρεις το username του διαχειριστή (ας υποθέσουμε ότι είναι `admin`), μπορείς να αλλάξεις τον κωδικό πρόσβασης με την εντολή `user:resetpassword`:

    ```bash
    php occ user:resetpassword admin
    ```
    Το σύστημα θα σου ζητήσει να πληκτρολογήσεις τον νέο κωδικό πρόσβασης δύο φορές.

    Μια απεικόνιση της διαδικασίας.
    

4.  **Έξοδος από το container:**
    Όταν τελειώσεις, μπορείς να βγεις από το shell του container πληκτρολογώντας `exit`:

    ```bash
    exit
    ```

**Σημαντικές Σημειώσεις:**

*   **Ασφάλεια Κωδικών:** Επειδή είναι δοκιμαστικός server, ίσως να μην έχεις αυστηρές απαιτήσεις για τον κωδικό, αλλά σε έναν παραγωγικό server είναι ζωτικής σημασίας να χρησιμοποιείς ισχυρούς και μοναδικούς κωδικούς πρόσβασης.
*   **Username:** Βεβαιώσου ότι χρησιμοποιείς το σωστό username του διαχειριστή. Συνήθως είναι `admin`, αλλά μπορεί να το έχεις αλλάξει κατά την αρχική εγκατάσταση.
*   **Docker Compose:** Επειδή χρησιμοποιείς `docker-compose`, οι εντολές `docker exec` είναι ο σωστός τρόπος να αλληλεπιδράσεις με τα services.

Με αυτά τα βήματα, θα μπορείς να αλλάξεις τον κωδικό πρόσβασης του διαχειριστή του Nextcloud μέσω SSH.
